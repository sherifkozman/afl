# Prototype: agentfail.html

This is the original concept prototype used to communicate the "Agent Failure Mode" idea:
- agents tend to keep answering even when they should stop,
- a harness that triggers a controlled failure state can reduce hallucinations.

The simulation inside is illustrative; it is not empirical measurement.

The rest of this repo turns that concept into a concrete agent runtime hook harness with a measurable test plan.
